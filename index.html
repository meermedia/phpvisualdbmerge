Welcome to PHP Visual DB Merge.

This tool will help you two merge two databases (for example a deployment and production database). Unlike other export/import tools or mysqldump commands, this tool will show you exactly which tables there are, match them up side by side, and let you select which table will be taken from which side. You can also skip certain tables or just use empty tables (avoids having to clear cache tables afterwards).

If you are a Drupal user, you will be happy to know that I have pre-installed some presets for Drupal 6 and Drupal 7(8), and also an intelligent 'auto' mode. The auto-mode should also work for WordPress websites.

Simply download the zip-file or clone our git-repository by using the following command:

$ cd your_web_root
$ git clone https://github.com/meermedia/phpvisualdbmerge.git .
$ git pull
$ cp sample.config.php config.php
$ vi config.php (type :wq to write and exit)
